Query: what is greedy algorithm

Answer: A greedy algorithm is an optimization algorithm that solves an optimization problem by choosing the locally optimal solution at each step, with the hope of finding a global optimum. It considers exactly one decomposition of the problem and explores nodes in a graph using best-first search.
